# Poker Trust Simulation

Multi-agent trust dynamics in 8-player Limit Texas Hold'em. Phase 1: 8
rule-based AI agents (no ML) play 10 000 hands with a Bayesian trust model.
See `Claude_Code_Implementation_Prompt.md` for the full build spec and
`The_Eight_Archetypes_Specification.docx` for the archetype definitions.

## Quick start

```bash
python3 -m pip install -r requirements.txt
python3 run_tests.py --stage all      # run every stage's tests
python3 run_demo.py                   # generate visualizer/data.js
```

Then open `visualizer/poker_table.html` in any modern browser. Re-run
`run_demo.py` after each stage to update the viewer with the latest agents.

### Demo flags

```bash
python3 run_demo.py --stage 2 --hands 20 --seed 42 --output visualizer/data.js
```

## Layout

```
Poker_trust/
├── config.py                    # central parameters (hand strength, blinds, bet sizes)
├── engine/
│   ├── deck.py                  # seeded Deck over treys.Card ints
│   ├── evaluator.py             # get_hand_strength (preflop lookup + Monte Carlo)
│   ├── actions.py               # ActionType enum + ActionRecord
│   ├── game.py                  # GameState + Hand (one hand of limit hold'em)
│   └── table.py                 # 8-seat Table with rebuys and dealer rotation
├── agents/
│   └── dummy_agent.py           # DummyAgent / FolderAgent / RaiserAgent (scripted)
├── data/
│   └── visualizer_export.py     # played hand → viewer JSON
├── visualizer/
│   ├── poker_table.html         # self-contained replay viewer
│   └── data.js                  # generated by run_demo.py (gitignored)
├── preflop_lookup.py            # 169-hand preflop bucketing (spec file)
├── archetype_params.py          # per-round agent parameters (spec file)
├── test_cases.py                # canonical test suites (spec file)
├── stage_extras.py              # real assertions layered onto test_cases
├── run_tests.py                 # stage-aware test runner
└── run_demo.py                  # stage-aware demo exporter
```
